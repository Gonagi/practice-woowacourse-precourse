# 자동차 경주

## 기능

### 게임

- 게임을 생성한다
- 우승자를 생성한다
- 게임을 플레이한다

### 자동차

- 자동차를 생성한다
- 움직인다
- 현재 위치의 인덱스를 반환한다

### 입력

- 자동차 이름을 입력받는다
- 시도 횟수를 입력받는다

### 출력

- 차수별 실행 결과를 출력한다
- 우승자를 출력한다

### 랜덤값 생성기

- 랜덤값을 생성한다
    - 1 ~ 9 사이의 랜덤 양수가 생성된다

---

## 오류 상황

### 자동차

- 자동차 이름 문자열에 알파벳 콤마가 아닌 다른 문자가 있을때는 오류가 발생한다

### 랜덤값 생성기

- 범위를 벗어난 숫자가 생성되면 오류가 발생한다
